# Hosty
Hosty is a hotel management system developed as a RESTful Web API using ASP.NET Core, designed with Clean Architecture. The project focuses on building a maintainable, scalable, and well-structured backend, emphasizing separation of concerns and a rich domain model. The system manages core hotel operations such as hotels, rooms, guests, reservations, and payments.

Key concepts and practices:

Clean Architecture (Domain, Application, Infrastructure, Persistence) Domain-Driven Design (Entities, Value Objects, Domain Rules) Repository and Unit of Work patterns ASP.NET Core Web API Entity Framework Core Dependency Injection

This project was created to demonstrate architectural decision-making, backend design skills, and best practices for enterprise-level applications.
